- Pin any floating pane so it stays on top even when focus moves to another pane
- Pin a floating pane with a mouse click on the pin icon or with the keyboard shortcut Ctrl p then i
- Great for keeping a reference pane, log tail, or monitoring output always visible while working in other panes
