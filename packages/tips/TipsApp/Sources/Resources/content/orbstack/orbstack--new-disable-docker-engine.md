- OrbStack v2.0.3 added the option to disable Docker Engine if you only use Linux machines and do not need containers
- Disabling Docker Engine reduces resource usage when you only use OrbStack for Linux VM functionality
- Toggle Docker Engine on or off from OrbStack settings without uninstalling or reinstalling anything
