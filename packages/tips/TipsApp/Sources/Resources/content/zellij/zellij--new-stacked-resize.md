- When resizing panes, Zellij automatically stacks them with their neighbors to maximize visible space while keeping all pane titles accessible
- Navigate to any stacked pane quickly using the keyboard or mouse even when it is not the focused pane
- Disable this behavior with stacked_resize false in your configuration if you prefer manual resize behavior
