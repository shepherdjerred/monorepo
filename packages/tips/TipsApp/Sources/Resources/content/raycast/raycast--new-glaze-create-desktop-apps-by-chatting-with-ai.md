- Glaze is a new Raycast tool that lets you create native Mac desktop apps just by describing what you want in plain language
- Apps built with Glaze run locally and support keyboard shortcuts, menu bar integration, file system access, and background processes
- If the result isn't right, just keep chatting to refine it — no coding knowledge required
- Glaze has a public marketplace and private team stores where you can discover, install, and customize existing apps
