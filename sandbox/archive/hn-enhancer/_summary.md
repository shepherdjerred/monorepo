HN Enhancer is a Chrome extension that improves Hacker News browsing by filtering negativity, hiding selected users, and flagging replies directly in comment threads. The codebase uses Bun-based tooling with Vite, TypeScript modules, Zod validation, and Chrome extension APIs defined in `manifest.json` to build and ship the extension. A background alarm worker in `src/background/index.ts` polls the Hacker News API for new replies, syncs cached state, and powers the reply-count badge rendered by the content script.
