## Quick Install

```bash
curl -fsSL https://github.com/shepherdjerred/monorepo/releases/latest/download/clauderon-linux-x86_64 -o clauderon
chmod +x clauderon && sudo mv clauderon /usr/local/bin/
clauderon daemon
```
