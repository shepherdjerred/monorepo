## Navigation

- `Command-Shift-G` - Jump directly to a folder.
- `Command-Option-S` - Toggle the sidebar.
- `Command-1` - Switch to icon view.

## File Management

- `Space` - Preview the selected file with Quick Look.
- `Command-Delete` - Move the selected item to the Trash.
- `Command-D` - Duplicate the selected item.
