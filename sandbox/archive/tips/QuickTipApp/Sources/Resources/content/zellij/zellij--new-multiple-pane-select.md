- Select multiple panes at once with Alt + left-mouse-click on each pane, or press Alt p to enter multi-select mode
- Perform bulk operations on selected panes: stack them, break them to a new tab, float them, and more
- Mouse hover effects are included as part of this feature; disable both with advanced_mouse_actions false in config
