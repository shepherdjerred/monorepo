- Swish 1.11 added an option to use macOS Sequoia's native window tiling for 2x2 snapping.
- When enabled, snapping a window triggers the system's built-in tiling instead of Swish's own placement logic.
- Grid spacing is automatically synced between Swish and Sequoia Window Tiling when this mode is active.
