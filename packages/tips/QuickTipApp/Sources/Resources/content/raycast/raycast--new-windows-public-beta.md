- Raycast is now available on Windows in public beta, bringing instant app launching, clipboard history, window management, and system-wide file search to Windows
- Raycast for Windows built its own custom file indexing system from scratch since Windows lacks a system-wide index that meets its performance standards
- Most extensions built with React and TypeScript work on both Mac and Windows automatically, giving Windows users access to thousands of community extensions immediately
- During the public beta, Quick AI is free without a subscription and powered by GPT-4o mini — type a question and hit Tab to get an answer with citations
