**Summary**
- Siphon Vue is a single-page client that replaces Harding University's Pipeline navigation with a modern search-driven interface linking students directly to financial aid, housing, and coursework resources.
- It is built with Vue.js, Vue Router, vuex-router-sync, Vue Resource, PureCSS, and Font Awesome, bundled through a Babel/Webpack toolchain defined in `package.json` and tracked by Travis CI.
- The app centralizes navigation through a weighted fuzzy-search configuration in `src/search.js` and route-driven views (`src/views/dashboard.vue`, `src/views/content-container.vue`, `src/views/settings.vue`) orchestrated by `src/router.js`.
