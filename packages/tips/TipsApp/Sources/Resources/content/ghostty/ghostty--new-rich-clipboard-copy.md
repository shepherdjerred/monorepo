- Copy terminal output in multiple formats at once — plain text, HTML, and VT escape sequences — all stored on the clipboard simultaneously.
- Paste into a rich-text editor to get formatted output with colors preserved; paste into a terminal to get raw escape sequences.
- Configure the `copy_to_clipboard` binding parameter to control which formats are included.
