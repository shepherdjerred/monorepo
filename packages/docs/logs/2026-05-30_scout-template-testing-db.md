# Scout Template Testing DB

## Status

Complete

## Summary

Answered what Scout for LoL's committed test template database is for.

The template lives at `packages/scout-for-lol/packages/backend/src/testing/template.db`. It is generated by `packages/scout-for-lol/packages/backend/scripts/generate-test-template-db.ts` via `bun run generate:test-template`, which is also run by the backend `db:generate` script.

The generated SQLite database contains the migrated schema plus seeded `Season` rows. Integration tests call `createTestDatabase(...)`, which copies this template into a per-test temporary `test.db` and constructs a Prisma client against that copy. This avoids running `prisma db push` or migrations for every test suite, making test setup much faster and avoiding prior Bun/Prisma setup instability.
