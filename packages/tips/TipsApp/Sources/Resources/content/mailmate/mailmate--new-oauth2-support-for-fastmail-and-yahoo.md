- MailMate 2.0 beta added OAuth2 authentication support for Fastmail and Yahoo accounts.
- OAuth2 is enabled automatically for existing Fastmail and Yahoo accounts — no manual reconfiguration needed.
- OAuth2 eliminates the need to store your password directly and uses secure token-based authentication instead.
