- Firefox 148 added support for the Sanitizer API, a web security standard that lets web developers safely insert untrusted HTML into a page by stripping out dangerous scripts before they execute.
- Websites can use element.setHTML() to sanitize user-generated content and block cross-site scripting (XSS) attacks at the browser level.
