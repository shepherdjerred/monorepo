- New `set_surface_title:<title>` and `set_tab_title:<title>` binding actions let you set titles directly from a keybind.
- Define keybinds to instantly switch between named contexts: `keybind = ctrl+shift+1=set_tab_title:Server`
- Combine with key tables to build a full title-management workflow triggered by a single leader key.
