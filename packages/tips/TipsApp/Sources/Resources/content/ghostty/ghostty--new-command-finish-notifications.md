- Get notified when a long-running command finishes with the new `notify-on-command-finish` configuration option.
- Set `notify-on-command-finish-after = 10` to only notify for commands that ran longer than 10 seconds.
- Control the notification action with `notify-on-command-finish-action` — useful for staying focused on other work.
