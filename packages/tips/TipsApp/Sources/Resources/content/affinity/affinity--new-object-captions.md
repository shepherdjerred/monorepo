- Object captions can now be attached to any placed image or object directly in the Layout Studio
- Position captions on any edge of the object with adjustable offsets for precise placement
- Captions can autonumber using built-in or custom counters, ideal for figures in long documents
- You can insert image metadata fields into captions and use them as targets for cross-references
