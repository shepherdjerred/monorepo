- Obsidian 1.12 added a built-in CLI so you can control Obsidian directly from your terminal
- Use it to open notes, run commands, create daily notes, and more without touching the mouse
- The CLI supports autocompletion for commands that use the id= parameter
- Run obsidian --help or obsidian help <command> to see available commands and their options
- Useful for scripting workflows, automation, and integrating Obsidian with other terminal tools
