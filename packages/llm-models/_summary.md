@shepherdjerred/llm-models is a catalog package that centralizes pricing and capability metadata for the team’s active LLM models so every language client can rely on the same definitions. Its TypeScript implementation in `src/index.ts` loads `catalog.json`, validates it with Zod, and emits typed accessors, while companion Python consumers validate the same data through Pydantic. A single `catalog.json` file serves as the cross-language source of truth, enforcing consistent schemas via `catalog.schema.json` and keeping token and image pricing units aligned.
