- Request a Copilot code review when creating a PR: gh pr create --reviewer @copilot
- Add Copilot as a reviewer to an existing PR: gh pr edit --add-reviewer @copilot
- Copilot also appears in the interactive reviewer search prompt when using gh pr create or gh pr edit without flags
