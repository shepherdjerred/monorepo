- You can now set a custom display name for your tailnet in Settings > General in the admin console, replacing your domain or email address in the client UI and CLI
- The display name appears across the admin console, client applications, and the CLI, making it easier to identify the right tailnet when managing multiple networks
- A Tailnet ID field was also added to Settings > General for use in the tailnetId field of Tailscale API calls, replacing the need to use your domain name directly
