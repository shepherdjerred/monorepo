This repository bundles custom Dev Container features that let developers install opinionated tooling inside containers, managed via the Dev Container feature spec. The Earthly feature is implemented with an Earthfile and `devcontainer features test --skip-autogenerated` workflow, including a shell installer that fetches the correct Earthly binary for the detected CPU architecture. After installation it can immediately run `earthly bootstrap --with-autocomplete` to wire up command completion alongside the binary in `/usr/local/bin`.
