- Zellij 0.42 introduced a new theme definition spec that lets you style generic UI components rather than mapping raw terminal colors
- The new spec extends theming to user plugins as well, so plugin authors can match the overall Zellij look
- See the updated theme documentation at zellij.dev/documentation/themes.html for the new format
