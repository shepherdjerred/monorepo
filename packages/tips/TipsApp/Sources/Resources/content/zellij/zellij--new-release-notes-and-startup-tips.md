- On first run after an upgrade, Zellij displays the release notes for the current version automatically
- On subsequent runs, a random useful tip is shown at startup to help you discover features
- Disable release notes with show_release_notes false or tips with show_startup_tips false in your config; browse both anytime with Ctrl o then a
