- Chat Branching lets you create alternate conversation paths from any point in an AI chat, so you can explore different directions without losing your original thread
- Press Cmd Shift B to branch from the current message, or right-click and choose Branch Chat
- Multiple branches can continue processing in the background while you switch between them
- Find Chat Branching under AI experiments in Settings
