- MailMate 2.0 beta added a Move to Mailbox toolbar button you can add to the message list toolbar.
- Right-click the toolbar and choose Customize Toolbar to add the Move to Mailbox button.
- Click it to quickly move selected messages to any mailbox without using the menu or keyboard shortcut.
