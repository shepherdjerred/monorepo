- Create a new Copilot coding agent task from the terminal: gh agent-task create "refactor the auth module"
- List all your agent tasks: gh agent-task list
- Stream the live log of a running task: gh agent-task view 1234 --log --follow
- Short aliases gh agent, gh agents, and gh agent-tasks are all available
