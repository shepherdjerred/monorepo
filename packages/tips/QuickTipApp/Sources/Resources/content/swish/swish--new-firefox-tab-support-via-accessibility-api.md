- Swish 1.13.1 added tab gesture support for Firefox now that it properly implements the Accessibility API.
- You can swipe on Firefox tabs to close, reorder, or switch between them using Swish gestures.
- Gestures still work outside the tab bar or by holding the Super Modifier if the tab bar is scrollable.
