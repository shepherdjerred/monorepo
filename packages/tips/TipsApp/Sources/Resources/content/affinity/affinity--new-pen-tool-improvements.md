- While drawing with the Pen Tool, press P to cancel drawing the current curve without deselecting it
- Alt-click any node to convert it to a sharp node, or Alt-drag to mirror the control handle length
- Rubber Band Mode, which previews the next curve segment before you click, is now enabled by default
- All Pen Tool settings are now remembered across documents and sessions so you can skip setup each time
- Join Curves is now available directly from the Vector menu for faster path operations
