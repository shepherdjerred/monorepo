- The File Explorer now supports copy and paste with Ctrl-C and Ctrl-V (Cmd-C and Cmd-V on Mac)
- Use Alt-click to add previous items to your selection for multi-file operations
- You can also drag results from the Quick Switcher directly into the editor or file explorer
