- Ghostty 1.2.0 adds Apple Shortcuts integration on macOS, letting you automate terminal actions from Shortcuts or Siri.
- Combine with the existing AppleScript support (added in 1.3.0) for powerful automation workflows.
- Undo/redo is now supported for all terminal closure operations — accidentally closed a tab? Undo it.
