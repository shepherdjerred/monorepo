- Convert any pixel selection into a fully editable vector curve using Vector menu > Convert to Curves
- Works with any selection method including the Object Selection tool, making it ideal for turning photo subjects into clean vector artwork
- Great for creating silhouettes, logo-style graphics, and cut-outs without manual tracing
- The resulting curve is fully editable and can be used as a vector mask
