Discord Plays Core orchestrates streaming and observability for interactive Discord game bots such as Pokémon and Mario Kart, handling tracing, metrics, audio transport, and bot bootstrapping. Built with Bun, TypeScript, and frameworks including Discord.js, Express, OpenTelemetry, Sentry, and prom-client, it provides structured logging and telemetry plumbing. Its package-level architecture exposes each module through dedicated subpath exports so games can import shared observability and streaming primitives while reusing pooled selfbot factories and Go-Live heuristics.
