- The plugin system is now available, letting you extend IINA's functionality with JavaScript plugins.
- Install plugins from the Plugins menu, or browse the plugin library at iina.io/plugins.
- Plugins can add new UI, react to playback events, and interact with mpv through a JavaScript API.
