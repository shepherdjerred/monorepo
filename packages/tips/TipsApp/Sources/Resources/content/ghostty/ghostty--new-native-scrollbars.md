- Ghostty 1.3.0 adds native overlay-style scrollbars via the new `scrollbar` configuration option.
- Set `scrollbar = system` (the default) to match your OS scrollbar style, or `scrollbar = never` to hide them entirely.
- Scrollbars support standard interactions like clicking and dragging on both macOS and GTK.
