This repository bundles custom Dev Container features for containers.dev, with each feature maintained under src/. The project uses Dev Container tooling plus Earthly via the Earthfile and shell-based installers such as src/earthly/install.sh to automate feature setup. A notable capability is the Earthly feature script that selects the correct architecture, installs the Earthly binary into /usr/local/bin, and bootstraps autocomplete when enabled.
