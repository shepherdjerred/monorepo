- Use `key-remap` in your config to remap physical keys before keybind processing — for example, swap Caps Lock and Escape at the terminal level.
- Use `clipboard-codepoint-map` to substitute specific Unicode codepoints when pasting from the clipboard.
- Use `selection-word-chars` to customize which characters are considered part of a word when double-clicking to select.
