- The visual policy editor (GA) lets you create and manage your tailnet access policy file using a graphical interface in the admin console instead of writing raw HuJSON
- You can define ACL rules, grants, and SSH access policies by selecting users, groups, and devices from drop-downs rather than memorizing policy file syntax
- The visual editor and the raw HuJSON editor stay in sync, so you can switch between them at any time without losing your configuration
