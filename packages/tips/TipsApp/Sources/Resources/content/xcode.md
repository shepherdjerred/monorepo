## Navigation

- `Command-Shift-O` - Open Quickly to find any file or symbol.
- `Control-6` - Show the issue navigator.
- `Command-J` - Reveal the current file in the project navigator.

## Editing

- `Control-I` - Re-indent the selected code.
- `Command-/` - Toggle line comments.
- `Command-Option-[` - Shift selected code left.
