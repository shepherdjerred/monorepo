Siphon Vue is a single-page app that modernizes navigation of Harding University’s Pipeline resources with a streamlined dashboard and curated external shortcuts (`README.md:5`). It runs on Vue 2 with Vue Router, Vuex synchronization, Vue Resource, PureCSS, Font Awesome, and vue-analytics for Google Analytics tracking (`src/app.js:1`, `src/analytics.js:1`). Routed views for the dashboard, content pages, and settings marry a configurable fuzzy search weighting system to quickly surface key Pipeline destinations (`src/router.js:1`, `src/search.js:1`).
