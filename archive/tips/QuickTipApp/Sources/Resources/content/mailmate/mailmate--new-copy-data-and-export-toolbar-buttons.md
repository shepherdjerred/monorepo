- MailMate 2.0 beta added optional toolbar buttons for exporting and copying email data.
- The Export Data button replaces the old Save Attachments button and supports more flexible export options.
- The Copy Data button copies message content or attachments to the clipboard with configurable format settings.
- Hold Option when clicking either button to skip the settings dialog and reuse your last settings.
